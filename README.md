# ⚾ KAGURA B.C. Score Manager (野球チームスコア管理・分析アプリ)

## 📖 概要
草野球チーム「KAGURA」（2015年設立）の活動における**スコア管理とデータ分析**を行うためのWebアプリケーションです。
従来、紙やExcelで行っていた煩雑な成績管理を自動化し、チームの強化方針や采配に活かすためのデータ分析基盤として開発しました。

## 🚀 主な機能

### 1. 成績入力・管理
* **打撃成績入力**: スタメン登録、打席ごとの結果入力（プルダウン選択）、試合中のスコアボード自動更新。
* **投手成績入力**: 投球回、失点、自責点、奪三振などの記録。
* **柔軟な入力モード**: 試合直後の「詳細入力」と、過去データを一括登録する「まとめ入力」の両方に対応。

### 2. データ分析・可視化
* **チーム成績ダッシュボード**: 勝率、平均得点/失点、チーム打率/防御率などを一目で確認。
* **個人成績ランキング**: 規定打席・規定投球回を考慮したランキング（打率、本塁打、OPS、防御率、WHIPなど）をメダルアイコンで可視化。
* **歴代記録**: シーズンハイ（最高記録）や通算成績の自動集計。
* **年度別推移**: 選手の成長を時系列で追跡。

### 3. UI/UX
* **スコアボード表示**: 実際の野球中継のようなスコアボードを動的に生成。
* **レスポンシブ対応**: スマートフォン（グラウンドでの入力）とPC（自宅での分析）の両方での操作を考慮。

## 🛠 技術スタック

* **言語**: Python 3.13
* **フレームワーク**: Streamlit (Frontend/Backend)
* **データ処理**: Pandas (データ加工、集計、スタッツ計算)
* **データベース**: Google Sheets API (簡易DBとして使用、リアルタイム更新)
* **バージョン管理**: Git / GitHub

## 🏗️ アーキテクチャと設計の工夫

初期開発時は単一ファイル(`app.py`)でしたが、機能拡張に伴う保守性の低下を防ぐため、**機能単位でのモジュール分割（リファクタリング）**を実施しました。

### ディレクトリ構成
```text
baseball_app/
├── app.py                # エントリーポイント（ルーティング制御）
├── config/               # 設定ファイル（定数、選手リスト、大会名など）
├── utils/                # 共通ロジック（DB接続、UIコンポーネント、計算ロジック）
└── views/                # 各画面のプレゼンテーション層
    ├── batting.py        # 打撃入力画面
    ├── pitching.py       # 投手入力画面
    ├── team_stats.py     # チーム成績分析
    ├── personal_stats.py # 個人成績・ランキング
    └── edit_data.py      # データ修正機能

こだわりポイント
保守性の向上: 画面ごとのロジックを views に分離し、メインファイルはルーティングに専念させることで、コードの見通しを改善。

拡張性: 選手名や球場リストを config/settings.py に外出しすることで、メンバー増減や環境変化にコード修正なしで対応可能に。

データ整合性: 詳細入力（1打席ごと）とまとめ入力（試合ごと）という異なる粒度のデータを、Pandasで統合・正規化して集計するロジックを実装。

📊 データ分析のアプローチ (Data Analysis)
単なる記録だけでなく、以下のような指標を自動計算し、チーム戦略に活用しています。

OPS (On-base Plus Slugging): 打者の総合的な貢献度を測るため、出塁率と長打率から算出。

WHIP (Walks plus Hits per Inning Pitched): 投手の安定感を測るため、1イニングあたりに出した走者の数を算出。

守備指標: 守備機会と失策数から守備率を算出。

💻 ローカルでの実行方法
# リポジトリのクローン
git clone [リポジトリURL]

# 依存ライブラリのインストール
pip install -r requirements.txt

# アプリケーションの起動
streamlit run app.py

※ 実行には .streamlit/secrets.toml にGoogle Sheets APIの接続情報が必要です。
